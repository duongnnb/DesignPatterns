# Design Patterns

Discover the modern implementation of design patterns with C# and .NET
## Installation

cd to destination folder

```bash
dotnet build
dotnet run
```
ex:
```bash
cd C:\source\repos\DesignPatterns\01. SOLID\DIP
dotnet build
dotnet run
```
